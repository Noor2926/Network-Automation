Port Scanner Help

What is a Port Scanner?

A port scanner is a tool that checks which ports are open on a target IP address. Open ports can indicate running services that might be accessible over the network.

How to Use

  1. Enter the target IP address you want to scan
  2. Select a port range (1-1024 for common ports, or custom range)
  3. Adjust timeout and thread settings if needed
  4. Click "Start Scan" to begin a full scan or "Quick Scan" for common ports only
  5. View results in the table below when the scan completes

Common Ports

20, 21 - FTP (File Transfer)

22 - SSH (Secure Shell)

23 - Telnet

25 - SMTP (Email)

53 - DNS (Domain Name System)

80 - HTTP (Web)

443 - HTTPS (Secure Web)

3306 - MySQL Database

3389 - RDP (Remote Desktop)

8080, 8443 - Alternative Web Ports

Security Considerations

  • Only scan networks you have permission to scan
  • Port scanning can be detected by security systems
  • Some services may be vulnerable if exposed to the internet
  • Consider closing unnecessary ports for better security
