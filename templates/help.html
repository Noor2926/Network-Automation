Help & Troubleshooting

Getting Started

Welcome to the Network Scanner application. This tool helps you discover and analyze devices on your network, scan for open ports, and identify potential vulnerabilities.

Main Features

  • Network Discovery - Scan your network to find all connected devices
  • Port Scanning - Check which ports are open on specific devices
  • Vulnerability Scanning - Identify potential security issues
  • Traffic Monitoring - Analyze network traffic patterns

Troubleshooting

Npcap Installation Issues

This application uses Npcap for packet capture functionality. If you're experiencing issues with network scanning or traffic monitoring, follow these steps:

  1. Download Npcap from the official website: https://npcap.com
  2. Run the installer with administrator privileges
  3. During installation, make sure to select the following options:
    • Install Npcap in WinPcap API-compatible Mode
    • Support raw 802.11 traffic
  4. After installation, restart your computer
  5. Run this application with administrator privileges

If you still experience issues after following these steps, try the following:

  • Uninstall any existing WinPcap installation before installing Npcap
  • Check if your antivirus software is blocking the application
  • Verify that your network adapter is properly configured

Network Scanning Not Working

If network scanning is not working properly, check the following:

  • Make sure you're connected to a network
  • Verify that you have the correct IP range for your network
  • Increase the scan timeout value for slower networks
  • Check if your firewall is blocking the scan
  • Run the application with administrator privileges

Port Scanning Issues

If port scanning is not working correctly:

  • Make sure the target device is online and reachable
  • Try reducing the number of threads to avoid overwhelming your system
  • Increase the timeout value for more reliable results
  • Some networks may block port scanning activities
  • Check if your firewall is blocking outgoing connections

System Requirements

Software Requirements

  • Python 3.7 or higher
  • Npcap (for packet capture functionality)
  • Scapy library (for advanced network operations)
  • Modern web browser

Required Python Packages

  • Flask
  • Scapy (optional, but recommended)

Hardware Requirements

  • Any modern computer with network connectivity
  • Minimum 4GB RAM recommended
  • Network adapter compatible with Npcap

Administrator Privileges

This application requires administrator privileges to perform network scanning and packet capture operations.
